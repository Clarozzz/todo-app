## How to run the project locally
Make sure you have installed:
- Node.js
- npm
- Expo CLI

```bash
npm install expo-cli
git clone https://github.com/Clarozzz/todo-app.git
cd todo-app
npm install
npx expo start
```

- Physical device (Expo Go)
